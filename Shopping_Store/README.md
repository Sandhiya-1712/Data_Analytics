# Data_Analytics 

# Online shopping store - Data Analysis

Short Description:
    It is a simple data analysis project using pandas and Power BI used for visualisation. the data for visualisation connected from mysql data sources.
  Here basic manipulation of data like replacing null values, renaming columns is done by pandas library,
  mysql has been used to do sum numerical aggregation and to retrieve data based on specific criteria.

Tools used :

Python for data manipulation
  - Pandas
  - mysql

For visualisation
  - PowerBI
