# Customer Churn Analysis Dashboard (Power BI)

## Overview

A Basic power BI Report which analyzes customer churn behavior using Power BI and mysql to clean up the data. Power BI is used to identify key drivers impacting retention and revenue.

## Key Insights

* Customers with month-to-month contracts show highest churn
* Fiber optic users have higher churn rate
* High monthly charges correlate with increased churn risk

## Tools Used

* Power BI
* DAX
* Data Modeling

## Features

* Connected through Import mode from mysql
* Interactive multi-page dashboard
* Drillthrough (Customer-level insights)
* Dynamic filtering (slicers)
* KPI metrics for churn and revenue
